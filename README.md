# E-Commerce Frontend (React)

## Project Overview

This project is a simple e-commerce frontend application built using React.
It allows users to browse products, view product details, add products to the cart, and simulate a checkout process.

## Features

* Product listing from API
* Product detail page
* Add to cart functionality
* Remove from cart
* Checkout form
* Responsive product grid
* React Router navigation
* Context API for cart state

## Technologies Used

* React
* React Router
* Context API
* FakeStore API
* CSS

## Installation

Clone the repository:

git clone <repo-link>

Install dependencies:

npm install

Run the project:

npm start
